# Intranet

I've made an intranet for my school project using:
- PHP
- HTML
- CSS
- Docker
- MySQL
- Javascript

In this intranet i added multiple features like:
- Registration
- Login system
- Group permissions
- Projects overview
- Project detail
- Edit projects
- Company news
- Activity board
- Employee overview

## DDEV
